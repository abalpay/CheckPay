# CheckPay Font Provenance

This project self-hosts font binaries under `app/fonts` for CSP-safe usage with `next/font/local`.

## Files Committed

| File | Family | Weight | Style |
|---|---|---:|---|
| `app/fonts/dm-serif-display-regular.woff2` | DM Serif Display | 400 | normal |
| `app/fonts/general-sans-regular.woff2` | General Sans | 400 | normal |
| `app/fonts/general-sans-medium.woff2` | General Sans | 500 | normal |
| `app/fonts/general-sans-semibold.woff2` | General Sans | 600 | normal |
| `app/fonts/general-sans-bold.woff2` | General Sans | 700 | normal |
| `app/fonts/jetbrains-mono-regular.woff2` | JetBrains Mono | 400 | normal |
| `app/fonts/jetbrains-mono-medium.woff2` | JetBrains Mono | 500 | normal |

## Source URLs Used

- DM Serif Display CSS manifest:
  - `https://fonts.googleapis.com/css2?family=DM+Serif+Display:wght@400&display=swap`
- JetBrains Mono CSS manifest:
  - `https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;500&display=swap`
- General Sans CSS manifest:
  - `https://api.fontshare.com/v2/css?f[]=general-sans@400,500,600,700&display=swap`
